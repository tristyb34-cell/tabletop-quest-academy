# Unreal Engine 5.7 Reference

Comprehensive reference covering all editor settings, project settings, and new features in UE 5.7.

---

## What's New in 5.7

### Substrate (Production Ready)
- Modular material authoring and rendering framework
- Replaces the old shading model system
- Allows layering physically accurate material types: clear coat, metal, cloth, skin, hair
- Enable: Project Settings > Engine > Rendering > Substrate > Enable Substrate
- If you're following older tutorials that mention "Shading Model" dropdown on materials, Substrate replaces that system

### MegaLights (Beta)
- Allows many more dynamic shadow-casting lights in a scene
- Realistic soft shadows from area lights
- Improved noise reduction and performance over 5.6
- Supports directional lights, Niagara particle lights, translucency, hair strands
- Enable: Project Settings > Engine > Rendering > MegaLights > Enable MegaLights
- Great for the tabletop scene (warm lamp light with soft shadows from multiple candles)

### Nanite Foliage (Experimental)
- Nanite now supports grass, shrubs, and trees with incredible density
- Combines Nanite Assemblies, Nanite Skinning, and Nanite Voxels
- Enable: Project Settings > Engine > Rendering > Nanite > Enable Nanite Foliage (Experimental)
- Not needed for Phase 1-2 but useful later for outdoor environments

### Procedural Content Generation (Production Ready)
- PCG framework for populating environments with variety
- New PCG Editor Mode for visual editing
- Procedural Vegetation Editor (PVE) for foliage placement
- GPU compute for faster PCG processing
- Access: Window > PCG Editor

### Lumen Improvements
- Sharper reflections
- Better indirect lighting for interior spaces
- Improved performance
- Already active if you're using Lumen (which you should be)

### Animation Improvements
- Refactored Animation Mode
- Improved IK Retargeter (better when retargeting Mixamo animations to UE5 skeletons)
- New Skeletal Editor improvements
- Dependency View for animation asset visualization

### Chaos Physics
- Better destruction simulation
- Improved cloth simulation
- More stable collision detection

### AI Assistant (Built-in)
- Dedicated slide-out panel in the editor
- Ask questions about Unreal Engine
- Generate C++ code snippets
- Step-by-step guidance
- Access: the AI icon in the top toolbar or Window > AI Assistant

## Editor Preferences (Edit > Editor Preferences)

### General

#### Appearance
- Path: Edit > Editor Preferences > General > Appearance
- Theme: Dark or Light (use Dark)
- UI Scaling: adjust for high-DPI monitors
- Editor Font: change the font used in editor panels

#### Performance
- Path: Edit > Editor Preferences > General > Performance
- Show Frame Rate and Memory: displays FPS and memory in the editor corner
- Use Less CPU when in Background: True (saves resources when alt-tabbed)
- Editor Performance Mode: reduces visual quality in editor for better performance on weaker machines

#### Region and Language
- Path: Edit > Editor Preferences > General > Region and Language
- Editor Language: English
- Region: affects number formatting

### Loading and Saving
- Path: Edit > Editor Preferences > Loading and Saving
- Auto Save: Enable (set to every 5 minutes)
- Auto Save Maps: True
- Auto Save Content: True
- Number of auto saves to keep: 5
- Prompt before auto saving: False (less interruption)

### Source Control
- Path: Edit > Editor Preferences > Loading and Saving > Source Control
- Source Control: Enable
- Provider: Git (if using Git LFS for binary assets)
- Auto checkout on asset modification: True

### Level Editor

#### Viewport
- Path: Edit > Editor Preferences > Level Editor > Viewport
- Field of View: 90 (default, good for game dev)
- Camera Speed: adjust scroll speed in viewport
- Camera Speed scalar: multiplier for faster navigation in large levels
- Grid snap size: 10 units default, change based on your grid needs
- Rotation snap: 15 degrees default
- Scale snap: 0.25 default

#### Play
- Path: Edit > Editor Preferences > Level Editor > Play
- New Viewport Resolution: set default PIE resolution
- Enable Game Sounds in PIE: True
- Enable Game Sounds in Editor: False (less noise while working)

### Content Editors

#### Blueprint Editor
- Path: Edit > Editor Preferences > Content Editors > Blueprint Editor
- Show Access Specifier in MyBlueprint: True (shows public/private on variables)
- Auto-Cast Object Pin on Connection: True
- Save on Compile: Always (prevents losing work)

#### Material Editor
- Path: Edit > Editor Preferences > Content Editors > Material Editor
- Live Preview: True (see changes in real-time)
- Use new Material Editor with Substrate: True (for 5.7)

### Keyboard Shortcuts
- Path: Edit > Editor Preferences > Keyboard Shortcuts
- Fully customizable. Key ones to know:
  - G: Toggle game view in viewport
  - F: Focus on selected actor
  - End: Snap actor to ground
  - Ctrl+S: Save current level
  - Ctrl+Shift+S: Save all
  - Ctrl+Z: Undo
  - Ctrl+Y: Redo
  - W/E/R: Switch between Translate/Rotate/Scale
  - Alt+drag: Duplicate actor
  - Ctrl+Alt+F11: Live Coding compile
  - F8: Eject from player in PIE
  - Shift+F1: Get mouse cursor back in PIE

## Project Settings (Edit > Project Settings)

### Engine Section

#### General Settings
- Path: Edit > Project Settings > Engine > General Settings
- Project Name: your game name
- Company Name: your studio name
- Default Map: the level that loads first
- Default GameMode: your custom GameMode class

#### Rendering
- Path: Edit > Project Settings > Engine > Rendering
- Global Illumination:
  - Dynamic Global Illumination Method: Lumen (recommended for this project)
  - Reflection Method: Lumen
  - Shadow Map Method: Virtual Shadow Maps (works with Lumen)
- Nanite:
  - Enable Nanite: True (for high-detail meshes)
  - Enable Nanite Foliage: Experimental in 5.7
- Substrate:
  - Enable Substrate: True (new material system in 5.7)
- Anti-Aliasing Method: TSR (Temporal Super Resolution, best quality/performance ratio)
- Default settings:
  - Auto Exposure: True
  - Motion Blur: personal preference (disable for cleaner combat visuals)
  - Ambient Occlusion: True

#### Physics
- Path: Edit > Project Settings > Engine > Physics
- Physics Engine: Chaos (default in 5.7)
- Enable Async Physics: True (better performance)
- Gravity: -980 (default, Earth gravity in cm/s^2)
- Default Physics Material: set friction and restitution defaults

#### Navigation System
- Path: Edit > Project Settings > Engine > Navigation System
- Navigation Mesh settings for AI pathfinding
- Agent Radius: 35 (default, adjust for character size)
- Agent Height: 144 (default, adjust for character size)
- Cell Size: 19 (smaller = more accurate but slower to generate)
- Tile Size: 48

#### Collision
- Path: Edit > Project Settings > Engine > Collision
- Object Channels: create custom channels (e.g., Projectile, Interactable, AbilityTrace)
- Trace Channels: create custom trace channels for line traces
- Preset: create collision presets for common actor types

#### Input
- Path: Edit > Project Settings > Engine > Input
- Default classes for Player Input and Input Component
- Enhanced Input: Default Mapping Context (set your main input mapping)

### Game Section

#### Game Mode
- Path: Edit > Project Settings > Maps and Modes
- Default GameMode: your custom GameMode class
- Game Instance: your custom GameInstance class (persists between levels)
- Default Pawn: your player character class
- HUD Class: your custom HUD
- Player Controller: your custom player controller

#### Asset Manager
- Path: Edit > Project Settings > Game > Asset Manager
- Primary asset types: configure how assets load
- Important for streaming and memory management

### Plugins Section

#### Enhanced Input
- Path: Edit > Project Settings > Plugins > Enhanced Input
- Enable Enhanced Input: True (should be default in 5.7)
- Default Player Mappable Input Config

#### Gameplay Ability System
- Make sure the GameplayAbilities plugin is enabled
- Path: Edit > Plugins > search "Gameplay Abilities" > Enable

#### Online Subsystem
- For future multiplayer: configure which online service to use

## Recommended First-Time Setup Checklist

1. Edit > Editor Preferences > Loading and Saving: Enable auto-save every 5 minutes
2. Edit > Editor Preferences > General > Performance: Show Frame Rate and Memory = True
3. Edit > Project Settings > Engine > Rendering: Set Lumen for GI and Reflections
4. Edit > Project Settings > Engine > Rendering: Enable Nanite
5. Edit > Project Settings > Engine > Rendering: Enable Substrate
6. Edit > Project Settings > Engine > Collision: Create "Projectile" and "AbilityTrace" channels
7. Edit > Plugins: Enable GameplayAbilities, Enhanced Input (should be default)
8. Edit > Project Settings > Maps and Modes: Set your default GameMode
9. Edit > Editor Preferences > Keyboard Shortcuts: Review and customize
10. Viewport: Switch to Lit mode, enable Show > Navigation for NavMesh visibility
