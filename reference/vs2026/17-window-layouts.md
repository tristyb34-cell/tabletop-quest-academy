## Window Layouts

### Saving and Restoring Layouts

**Window > Save Window Layout** (Ctrl+Shift+1 through Ctrl+Shift+4 to save, Ctrl+Alt+1 through Ctrl+Alt+4 to apply)

| Action | Method |
|--------|--------|
| Save layout | Window > Save Window Layout, assign name |
| Apply layout | Window > Apply Window Layout > select saved layout |
| Manage layouts | Window > Manage Window Layouts (rename, delete) |
| Reset to default | Window > Reset Window Layout |

### Window Docking

- **Drag a window title bar** to dock it to any edge
- **Hold Ctrl while dragging** to prevent docking (keep floating)
- **Guide diamonds** appear while dragging to show docking positions
- **Double-click title bar** to toggle between docked and floating

### Tab Groups

| Action | Description |
|--------|-------------|
| New Horizontal Tab Group | Split the editor horizontally, documents in top/bottom groups |
| New Vertical Tab Group | Split the editor vertically, documents in left/right groups |
| Move to Next/Previous Tab Group | Move a document between groups |
| Close Tab Group | Close all documents in a group |

### Split Views

- **Window > Split** or drag the splitter handle at the top of the vertical scrollbar
- Creates two views of the same document
- Each view can scroll independently
- Useful for comparing different parts of a long file

### Floating Windows

- Any tool window can be undocked and floated
- Useful with multiple monitors: float Solution Explorer to a second screen
- Tool windows remember their floating position

### Auto-Hide

- Click the pin icon on a tool window title bar to auto-hide it
- The window collapses to a tab on the nearest edge
- Hover or click the tab to temporarily show the window
- Pin it again to keep it permanently visible
