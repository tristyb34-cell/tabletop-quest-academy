## Keyboard Shortcuts (Comprehensive Reference)

### General

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+P | Feature Search / All-In-One Search (new in VS 2026) |
| Ctrl+T or Ctrl+P | Go To All / Code Search |
| Ctrl+Shift+N | New Project |
| Ctrl+Shift+O | Open Project/Solution |
| Ctrl+N | New File |
| Ctrl+O | Open File |
| Ctrl+S | Save |
| Ctrl+Shift+S | Save All |
| Ctrl+F4 or Ctrl+W | Close Tab (Ctrl+W new in VS 2026) |
| Ctrl+Tab | Switch between open documents |
| Alt+F4 | Exit Visual Studio |
| Shift+Alt+Enter | Toggle Full Screen |
| F4 | Properties Window |

### Editing

| Shortcut | Action |
|----------|--------|
| Ctrl+C | Copy (copies entire line if nothing selected) |
| Ctrl+X | Cut (cuts entire line if nothing selected) |
| Ctrl+V | Paste |
| Shift+Alt+V | Adaptive Paste (Copilot-adjusted, new in VS 2026) |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+D | Duplicate line |
| Ctrl+L | Delete line |
| Ctrl+Shift+K | Delete line (alternative) |
| Ctrl+Enter | Insert line above |
| Ctrl+Shift+Enter | Insert line below |
| Alt+Up/Down | Move line up/down |
| Ctrl+U | Make lowercase |
| Ctrl+Shift+U | Make uppercase |
| Tab (with selection) | Indent selection |
| Shift+Tab | Outdent selection |
| Ctrl+K, Ctrl+C | Comment selection |
| Ctrl+K, Ctrl+U | Uncomment selection |
| Ctrl+/ | Toggle line comment (new in VS 2026) |
| Ctrl+K, Ctrl+D | Format document |
| Ctrl+K, Ctrl+F | Format selection |
| Ctrl+R, Ctrl+W | Toggle visible whitespace |
