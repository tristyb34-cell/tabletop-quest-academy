## Accessibility

### High Contrast Themes

VS 2026 includes extra contrast options designed for users with visual needs:

| Feature | Description |
|---------|-------------|
| High Contrast theme | Maximum contrast between text and background |
| Extra contrast editor options | Additional editor-specific contrast settings independent from the IDE theme |
| Increased text legibility | All built-in themes designed to minimize eye strain |

### Screen Reader Support

| Feature | Description |
|---------|-------------|
| Full MSAA/UIA support | Compatible with JAWS, NVDA, and Narrator |
| Live regions | Screen readers announce build results, error list changes, and notifications |
| Accessible controls | All dialogs, tool windows, and menus are keyboard-navigable |

### Keyboard Navigation

| Feature | Description |
|---------|-------------|
| Full keyboard access | Every feature is accessible via keyboard |
| Tab order | Logical tab order through all controls |
| Focus indicators | Clear visual focus indicators on all interactive elements |
| Ctrl+M, Ctrl+O | Collapse to definitions for overview without scrolling |

### Font and Spacing

| Feature | Setting Path |
|---------|-------------|
| Font size | Tools > Options > Environment > Fonts and Colors |
| Compact spacing in Solution Explorer | Tools > Options > Environment > Visual Experience > Use compact spacing |
| Line spacing | Adjustable through text editor settings |
