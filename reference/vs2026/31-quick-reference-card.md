## Quick Reference Card

### Most Important Settings for UE5 (Do These First)

| Setting | Path | Value |
|---------|------|-------|
| Color theme | Environment > General | Dark |
| Just My Code | Debugging > General | DISABLE |
| Edit and Continue | Debugging > C++ | DISABLE |
| Max Cached Translation Units | Text Editor > C/C++ > Advanced | 8 (if 32GB+ RAM) |
| Disable External Dependencies | Text Editor > C/C++ > Advanced | True |
| Parallel builds | Projects and Solutions > Build and Run | Match CPU core count |
| Only build startup project on Run | Projects and Solutions > Build and Run | True |
| Tab type | Text Editor > C/C++ > Formatting > Indentation | Tabs (size 4) |
| Sticky Scroll | Text Editor > General | Enable |
| CodeLens | Text Editor > All Languages > CodeLens | Disable |
