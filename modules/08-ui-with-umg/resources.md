## Video Tutorials

- **Ryan Laley, UE5 UMG Playlist**: Search "Ryan Laley UE5 UMG UI" on YouTube. Covers widget creation, layout panels, data binding, and full menu systems.
- **Matt Aspland, RPG UI Series**: Search "Matt Aspland UE5 RPG inventory" on YouTube. Step-by-step inventory, character sheets, and HUD creation for RPGs.
- **Gorka Games, UMG Tutorials**: Search "Gorka Games UE5 UMG" on YouTube. Clean, beginner-friendly tutorials on common UI patterns.
- **Unreal Engine Official, UMG Live Training**: https://dev.epicgames.com/community/unreal-engine/learning - filter by UI topic. The live training sessions walk through real projects.
- **Ali Elzoheiry, UE5 Dialogue System**: Search "Ali Elzoheiry UE5 dialogue system" for a branching dialogue implementation with UMG.

## UI Design for Games

- **Game UI Database**: https://www.gameuidatabase.com/ - a massive collection of screenshots from real game UIs, searchable by genre, element type, and style. Excellent reference for RPG interfaces.
- **Interface in Game**: https://interfaceingame.com/ - curated collection of video game UI screenshots with analysis. Great for studying how professional RPGs handle HUDs, menus, and combat UI.
- **Laws of UX**: https://lawsofux.com/ - core UX principles explained simply. Not game-specific but directly applicable (Fitt's Law, Hick's Law, etc.).
- **Material Design Color Tool**: https://material.io/resources/color/ - useful for building a consistent color palette for your UI theme.

## RPG-Specific UI References

- **Baldur's Gate 3 UI Breakdown**: Search "BG3 UI design" for analysis articles and videos. BG3 handles initiative, character sheets, inventory, and dialogue in ways very similar to what this project needs.
- **Divinity: Original Sin 2 UI**: Another excellent reference for turn-based RPG interfaces with party management, ability bars, and initiative tracking.
- **Solasta UI**: A DnD 5e video game with faithful implementation of character sheets, dice rolls, and turn order. Good reference for data-heavy RPG screens.

## Font Resources

- **Google Fonts**: https://fonts.google.com/ - free fonts. Look at "Cinzel" or "Almendra" for fantasy/medieval titles, and "Lato" or "Source Sans Pro" for readable body text.
- **Font Squirrel**: https://www.fontsquirrel.com/ - free commercial-use fonts with a generator for creating web/game font packages.
- **DaFont Fantasy Category**: https://www.dafont.com/theme.php?cat=401 - free fantasy and medieval display fonts. Check licenses before commercial use.

## Icon Resources

- **Game-icons.net**: https://game-icons.net/ - massive library of free game icons (swords, shields, potions, spells, etc.) in SVG format. Perfect for inventory and ability icons.
- **Kenney UI Pack**: https://kenney.nl/assets/ui-pack - free, clean UI element sprites (buttons, panels, sliders, checkboxes). Great for prototyping.
- **OpenGameArt UI Assets**: https://opengameart.org/art-search-advanced?field_art_type_tid[]=9 - community-created UI assets, free to use.

## Advanced Topics

- **Common UI Plugin (UE5)**: https://dev.epicgames.com/documentation/en-us/unreal-engine/common-ui-plugin-for-advanced-user-interfaces-in-unreal-engine - Epic's plugin for building platform-agnostic UI that works across mouse, gamepad, and touch. Worth exploring once your basic UI is working.
- **Slate (C++ UI)**: https://dev.epicgames.com/documentation/en-us/unreal-engine/slate-ui-framework-for-unreal-engine - the underlying C++ framework that UMG is built on. Only needed if you hit UMG's limitations and need custom widgets.
- **UMG Performance Best Practices**: Search "UE5 UMG optimization" for community guides on widget invalidation, pooling, and tick management.
